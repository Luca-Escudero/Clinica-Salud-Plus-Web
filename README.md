# 🏥 Clínica Salud+ - Sistema de Gestión de Turnos

## 📝 Descripción
**Clínica Salud+** es una plataforma web desarrollada para simplificar la reserva y administración de turnos médicos. El proyecto se centra en ofrecer una interfaz limpia, profesional y fácil de usar tanto para pacientes como para el personal administrativo.

## 🚀 Funcionalidades Clave
* **Reserva de Turnos:** Interfaz dinámica para seleccionar especialidades y horarios.
* **Gestión de Pacientes:** Formulario de registro y validación de datos.
* **Panel Administrativo:** Visualización y control de la agenda médica.
* **Diseño Responsive:** Optimizado para funcionar en computadoras, tablets y celulares.

## 🛠️ Tecnologías Utilizadas
* **Frontend:** HTML5, CSS3 y JavaScript.
* **Framework CSS:** **Bootstrap 5** (utilizado para el diseño de componentes y grid system).
* **Iconografía:** [Google Icons].
* **Lógica:** Manipulación del DOM y validaciones con JS.
